# 00-inbox/

**Generic unfiled graveyard.** For stuff that doesn't have an obvious home yet and doesn't fit any specific workflow.

Think: a PDF someone sent you, a `.tex` file you forgot why you saved, a screenshot you'll deal with later. Things that need a human decision before they belong anywhere.

## What goes here

- Random files that arrived without a workflow attached (PDFs, images, tex, etc.).
- Stuff awaiting a "do I keep this, file it, or delete it?" decision.
- One-off attachments not yet linked from any note.

## What does NOT go here

- **Stream-of-consciousness writing** → use `raw/dumps/`. That has a processing flow; this folder doesn't.
- **External papers/articles** to be integrated into the wiki → use `raw/incoming/`. The wiki-playbook flow processes those.
- **Daily journal-style entries** → use `10-daily/`.

## Lifecycle

Stuff here is meant to be temporary. Periodically (e.g., once a month) sweep through and decide:
- File it into the right folder, or
- Move it to `90-archive/` if it's done its job, or
- Delete it.

If something has been sitting here for a year, you probably don't need it.
